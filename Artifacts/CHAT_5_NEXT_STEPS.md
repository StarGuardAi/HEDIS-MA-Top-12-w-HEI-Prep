## üö® Troubleshooting

### Resume PDF Won't Generate

**Problem:** WeasyPrint dependencies missing on Windows

**Solution:** 
1. Use the simple generator that created HTML ‚úÖ
2. Print to PDF from browser (Ctrl+P ‚Üí Save as PDF)
3. Or install GTK3 Runtime: `winget install GTK3_Runtime`

### LinkedIn Post Not Getting Engagement

**Solutions:**
- Post at optimal times (Tuesday 8-10 AM, Wednesday 12-1 PM)
- Use all hashtags (10-15 relevant hashtags)
- Tag 2-3 relevant people (sparingly)
- Share in LinkedIn groups
- Respond quickly to comments

### GitHub Profile Not Showing README

**Problem:** README not in correct location

**Solution:**
1. Must be in `.github/profile/README.md`
2. Repository name must match your username exactly
3. Check the folder structure: repo ‚Üí `.github` ‚Üí `profile` ‚Üí `README.md`

### Can't Find LinkedIn Groups

**Solutions:**
- Search for keywords: "data science", "threat intelligence", etc.
- Sort by "Most relevant"
- Join groups with 10K+ members
- Participate actively once joined
