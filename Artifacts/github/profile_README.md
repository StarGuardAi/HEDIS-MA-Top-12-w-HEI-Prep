## Setup Instructions

### Step 1: Create Profile Repository

1. Go to GitHub.com and create a new repository
2. Repository name: **your-username**
3. Public repository
4. Add a README.md file

### Step 2: Create Profile File Structure

```bash
mkdir -p .github/profile
cd .github/profile
```

### Step 3: Add README.md

Copy the content above into `.github/profile/README.md`

### Step 4: Customize

- Update stats and achievements
- Add recent activity
