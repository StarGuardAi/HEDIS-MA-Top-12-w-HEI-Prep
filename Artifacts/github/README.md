## Quick Start

### Step 1: Update Personal Profile

1. Create a repository named `yourusername` (public)
2. Add `.github/profile/README.md`
3. Copy content from `profile_README.md`
4. Commit and push

**Time required:** 15 minutes

### Step 2: Update Organization Profile

1. Go to github.com/reichert-sentinel-ai
2. Create/update `sentinel-analytics` repository
3. Copy content from `organization_README.md`

**Time required:** 20 minutes

1. Go to GitHub profile
2. Click "Customize your pins"
3. Select 6 relevant repositories

1. Go to organization main page
2. Click "Customize your pins"
3. Select 6 repositories

**Time required:** 5 minutes

## GitHub Actions Badges

### CI/CD Status

Add to repository README:

```markdown
[![CI](https://github.com/user/repo/workflows/CI/badge.svg)](https://github.com/user/repo/actions)
```

### Example GitHub Actions Workflow

```yaml
# .github/workflows/ci.yml
name: CI

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-python@v4
        with:
          python-version: '3.11'
      - run: pip install -r requirements.txt
      - run: pytest
```

### Enable GitHub Pages for Portfolio

1. Go to repository Settings

3. Source: Deploy from a branch
4. Branch: main /docs
5. Save

1. Add CNAME file to repository
