## Quick Start

### Install Dependencies

```bash
pip install markdown weasyprint
```

**Windows:** Also install GTK3 runtime:
```powershell
winget install GTK3_Runtime
```

### Generate Resume

```bash
python resume_generator.py
```

This will create:
- `resume_output.pdf` - Professional PDF for applications
- `resume_output.html` - Preview in browser
- `resume_plain_text.txt` - Plain text for online forms

## LinkedIn Integration

The plain text format can be imported to LinkedIn:
1. Go to LinkedIn → Profile
2. Click "Add profile section" → Background → Resume
3. Upload `resume_plain_text.txt`
